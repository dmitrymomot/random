# random

Small and simple helper to generate "random" string.

## Usage

Installation:
```
go get -u github.com/dmitrymomot/random
```
Use
```golang
import "github.com/dmitrymomot/random"

str := random.String(16)
log.Println(str)
```
Output:
```
Cb0ajMig6N7l9Fzf
```
